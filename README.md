### ℹ️ Description

**Rappel2Makkah** is a web application designed to provide users with timely reminders and resources related to Islamic practices. It aims to assist users in staying connected with their faith through scheduled notifications and curated content.

- 🕋 Dourous: Various downloadable sounds/records dourous.
- 📚 Islamic Resources: Provides access to a collection of Islamic teachings.
- 🔔 Custom Reminders: Allows users to set personalized reminders.

---

## 🚀 Planned Features

- 📱 Mobile Application: Develop a mobile version for both iOS and Android platforms.
- 🌐 Multi-language Support: Offer the application in multiple languages to cater to a diverse user base.
- 📩 Email Notifications: Implement email alerts for important Islamic dates and events.
- 🔊 Allow users to download audio files.
- 📕 Transliteration of all records and download them.

# ✅ Getting Started

This project runs on 🅰️ Angular. Ensure you have Node.js and Docker installed on your system before proceeding with the installation.
Below are installation instructions for a Angular-based project.

### 💻 Installation

```bash
# Clone the repository
git clone https://github.com/nlabrazi/rappel2makkah.git
cd rappel2makkah

# Install dependencies
npm install

# Set up the environment variables
cp .env.example .env
# Update the .env file with your configuration

# Run the application
npm run start
```
